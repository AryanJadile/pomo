# PomeGuard Standards

## Cultivar Priority: Bhagwa
- All reference values, environmental tolerances, and phytochemical baseline data MUST default to the **Bhagwa** pomegranate cultivar.
- Do not use generic pomegranate baselines unless Bhagwa-specific data is completely unavailable.

## Measurement Units
- Phytochemical concentrations MUST be expressed in **mg/100g** (milligrams per 100 grams of fresh weight).
- No other unit (e.g., %, ppm) is acceptable for output unless explicitly requested for a non-standard calculation.

## Environmental Data Processing
- Expect inputs of Temperature (°C), Humidity (%), and Irradiance (W/m²).
- Adjust disease maps accounting for these environmental triggers.
